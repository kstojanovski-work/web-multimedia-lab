# Web Multimedia Lab

`web-multimedia-lab` is a unified web multimedia platform lab, created with Codex support.

## Purpose

- Real-time text communication
- Real-time audio/video communication
- Customer-agent assistive communication workflows
- Live sign-language recognition to text captions
- Text-to-sign video generation and delivery
- End-to-end experimentation for accessibility-focused communication systems

## Transmission methods

- WebSockets for bidirectional real-time signaling and messaging
- WebRTC for low-latency peer media streaming
- HTTP/REST for API operations, job management, and artifact access
- Background job queues for asynchronous media processing

## Libraries and frameworks

- FastAPI
- React (Vite)
- Uvicorn
- WebSocket APIs
- WebRTC APIs
- PyTorch
- MediaPipe
- Redis
- RQ (Redis Queue)
- SQLite
- Docker / Docker Compose
